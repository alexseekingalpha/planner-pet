Planner Pet: By Alexander Smallwood
Welcome to my project - Planner pet! 
Inspired by my struggles with ADHD I wanted to mkae a deivce to motivate others and I to complete tasks regardless of what the task is. 
Take homework for example. Homework is so hard to complete as there is no instant reward, you must wait to finish the problem to recieve the dopamine of completing such homework. 
To fix this I hypothesise that a digital pet will create a bond to the user motivating them to complete tasks to keep the pet alive! Like a tamagotchi!

The process is simple and fun:
Complete tasks and your pet earns XP which lets it grow through life stages, and unlocks cosmetics. Miss tasks and it regresses eventually going dormant.
To enhance attachment, each pet is generated from a random seed so everyone has a unique pet!

Hardware:
MCU: Esp32-S3-Wroom-1-N8
Display: st7789 240x250 tft spi 
RTC: DS3231 
Power: 500mah lipo whith BQ24040 charging via USBC, TLV75801 ldo to regualate the power to the MCU
Input: 4 button dpad with + 2 standalone buttons for 'select' and 'back'
All KiCad files will be uploaded (Using KiCad 9)
